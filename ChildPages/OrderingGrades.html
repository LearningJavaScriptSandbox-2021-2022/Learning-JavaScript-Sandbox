Ordering grades

Description of Assignment

  • Problem: calculate up to 5 test scores into a letter value
  • Test cases:
    • F: 1 - 59%
    • D: 60 - 69%
    • C: 70 - 79%
    • B: 80 - 89%
    • A: 90 - 100%
  • Note: zero% is not allowed
  • Note: calcualting (mean) means adding 5 scores and dividing by the number of scores
  • note: able to have less than 5 test scores and mean should still work

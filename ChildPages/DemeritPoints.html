Demerit points

Description of Assignment
  • Problem:Input a speed and calculate the Demerit Points
  • Note: Speed limit is 70 km per hour
  • Note: For every 5 km/h above the speed limit, add one more demerit point
  • Note: At any speed limit, 12 demerit points means license suspended
  • Test cases:
    • 68 km/h 0 points
    • 70 km/h 0 points
    • 71 km/h 0 points
    • 74 km/h 0 points
    • 75 km/h 1 points
    • 85 km/h 3 points
    • 102 km/h 6 points (round down)
    • 115 km/h 9 points
    • 130 km/h license is suspended
  • Coding hints
    • Math.floor(): function to remove any decimal points, no rounding
